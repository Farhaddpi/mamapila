## IMPORTANT — how to upload so images & videos show
Images did not appear before because the assets/ FOLDER was not uploaded.
1. Extract this zip on your computer.
2. github.com/farhaddpi/mamapila  →  Add file  →  Upload files.
3. DRAG the whole `assets` folder into the upload box (do NOT click
   "choose your files" — that only allows single files, not folders).
   Also drag index.html, fa.html, styles.css, script.js, README.md.
4. Wait until every file (especially the .mp4 videos) finishes, then Commit.
5. Wait ~1 minute and hard-refresh (Ctrl + Shift + R).
